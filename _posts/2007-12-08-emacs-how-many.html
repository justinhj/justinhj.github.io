--- layout: post title: 'emacs: how-many' date: '2007-12-08T15:18:00.000-08:00' author: Justin tags: modified_time: '2007-12-08T15:25:18.649-08:00' blogger_id: tag:blogger.com,1999:blog-4777243148323391813.post-3362163213750999415 blogger_orig_url: http://justinsboringpage.blogspot.com/2007/12/emacs-how-many.html --- Discovered another emacs command yesterday:

how-many

Returns the count of a regex following the point (cursor position).

For example

M-x how-many
\b\w

returns a word count from point to the end of the buffer.

Nice.
