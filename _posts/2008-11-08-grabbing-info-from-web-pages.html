--- layout: post title: Grabbing info from web pages date: '2008-11-08T22:03:00.001-08:00' author: Justin tags: modified_time: '2008-11-09T10:17:01.899-08:00' blogger_id: tag:blogger.com,1999:blog-4777243148323391813.post-4696422128723514955 blogger_orig_url: http://justinsboringpage.blogspot.com/2008/11/grabbing-info-from-web-pages.html --- This is how you use wget to grab a web page to stdout ...

wget -O - http://newsrss.bbc.co.uk/rss/newsonline_world_edition/front_page/rss.xml

Where -O species the output file, and the - specifies use stdout instead of a filename.
